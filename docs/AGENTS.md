# Agent Team

## Product Architect Agent
Owns product scope, feature prioritization, pricing model, and roadmap.

Tasks:
- Keep MVP lean
- Prevent feature creep
- Define free/pro/agency boundaries
- Review UX for conversion

## Frontend UI Agent
Owns layout, components, responsive design, and visual polish.

Tasks:
- Build dashboard UI
- Build public page templates
- Maintain mobile-first design
- Ensure accessibility basics

## Backend/Supabase Agent
Owns database, auth, policies, analytics inserts, and API routes.

Tasks:
- Implement Supabase client
- Validate RLS policies
- Create typed queries
- Protect dashboard routes

## Growth Agent
Owns positioning, landing page copy, onboarding, and niche expansion.

Tasks:
- Write homepage copy
- Create onboarding templates
- Generate launch campaigns
- Build niche-specific template ideas without hard-branding the product

## QA Agent
Owns testing, edge cases, and launch checklist.

Tasks:
- Test auth flows
- Test public pages
- Test link tracking
- Test mobile layout
- Check broken states
