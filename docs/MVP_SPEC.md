# MVP Spec: AI Bio Website Platform

## Positioning
Not a Linktree clone. A fast mini-website builder for creators, businesses, job seekers, consultants, artists, and local operators.

## Core Value Proposition
Create a professional bio website in minutes with links, lead capture, monetization blocks, QR code, and optional AI-generated copy.

## MVP Features

### Public Page
- Username route: `/u/[slug]`
- Profile image/logo
- Headline and short bio
- Social links
- Main buttons/links
- Service/product cards
- Contact CTA
- Optional booking link
- Optional newsletter link
- QR code
- SEO metadata

### Dashboard
- Login/signup
- Create/edit profile
- Add/edit/reorder links
- Toggle sections on/off
- Choose theme
- View basic analytics

### Analytics
- Page views
- Link clicks
- Top links
- Referrer tracking if available

### Admin/Owner
- See all users
- Manage reported pages later
- Feature flags later

## Free Plan
- 1 page

- Basic themes
- Platform branding
